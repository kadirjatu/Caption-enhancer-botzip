Create a complete production-ready Python project for Replit that integrates AI image and video upscaling directly into an existing Python Telegram bot.

Project Stack

* Python 3.11
* python-telegram-bot (async)
* FFmpeg
* Real-ESRGAN
* Pillow
* OpenCV
* aiofiles

Features

1. Image Upscaling

* Receive JPG, PNG and WEBP images from Telegram.
* Upscale using the official Real-ESRGAN pretrained models.
* Default upscale: 4x.
* Allow optional 2x or 4x scale.
* Preserve transparency.
* Return the processed image back to the user.

2. Video Upscaling

Support:

* MP4
* MOV
* MKV
* AVI

Workflow:

* Download Telegram video.
* Save temporarily.
* Extract frames using FFmpeg.
* Process frames sequentially with Real-ESRGAN.
* Merge frames back using FFmpeg.
* Copy original audio.
* Send the final video back to Telegram.

Support videos up to 5 minutes.

3. Performance

* Process frames sequentially to reduce RAM.
* Use asynchronous processing.
* Display progress updates in Telegram.
* Handle multiple jobs safely.
* Delete temporary files automatically after completion or failure.

4. Automatic Model Setup

If the official Real-ESRGAN repository is missing:

* Clone the latest stable release automatically.

If pretrained weights are missing:

* Download them automatically.

Store inside:

models/

If already downloaded:

* Skip downloading.

Cache models permanently.
